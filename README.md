# m_SerialBurning
03T 语音芯片烧录工具支持win/linux烧录</br>
win环境烧录附带GUI工具无需命令调用,如果串口烧录引导失败可以调整降低延迟参数(毫秒)</br>
03T新旧芯片在烧录上可能存在延迟不一样的情况需要根据情况适当修改源码!</br>
开发环境需要:vs2019 NET5</br>
